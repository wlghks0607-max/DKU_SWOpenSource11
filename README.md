# 점프 플랫포머 게임
1.프로젝트 개요
제안 내용: 선형적인 스테이지 클리어 방식의 점프 플랫포머 게임 개발

배경: 중력, 충돌 등 게임 물리 엔진의 기저 원리를 가장 직관적이고 상세하게 다루고 학습할 수 있는 최적의 주제로 판단함.

기대효과: 점프 플랫포머의 기초적인 옵션들을 넣어서 만든 오픈소스 게임이기 때문에, 여러 사람들이 편하게 변형 및 가공할 수 있게 도와줄 수 있다. 개발자들에게 여러 물리엔진과 관련된 내용에 대해서 학습하기 쉬운 사례가 될 수 있다. 

2.제공 기능

게임 시작 후 이동, 점프, 착지와 관련된 물리 상태 그룹을 만들어서 플레이어 캐릭터의물리 엔진을 완성하고, 그에 따른 게임 클리어와 게임 오버에 관련한 결과를 출력할 수 있게 만든 다이어그램이다. 

메뉴 화면 구성: 게임 시작키, 타이틀명, 레벨 선택, 저장, 불러오기 기능

사용자 시나리오: 플레이어는 메인 메뉴에서 게임을 시작해 첫 번째 레벨에 입장한다. 방향키를 꾹 누르자 캐릭터가 가속을 받으며 달려나가고, 점프 시 자연스러운 포물선 궤도를 그리며 이동한다. 찰나의 실수로 장애물에 부딪혀 게임 오버를 맞이하지만, 빠른 재시작을 통해 다시 도전을 할 수 있다. 마침내 모든 장애물을 피해 클리어 발판에 도착하면, 성취감을 주는 게임 클리어 연출과 함께 플레이 데이터가 저장된다.
