# 사용자 가이드 (User Guide)

**점프 플랫포머 게임 — 물리 엔진 학습을 위한 오픈소스 2D 플랫포머**

이 문서는 게임을 처음 실행하는 플레이어를 위한 안내서입니다. 설치부터 조작법, 게임 진행 방법까지 순서대로 설명합니다.

---

## 1. 게임 소개

방향키로 캐릭터를 움직이고 점프하여 장애물을 피하면서, 스테이지 끝의 **목표 발판(Goal)** 에 도달하면 클리어되는 선형 진행 방식의 2D 점프 플랫포머입니다. 이동 가속, 점프 포물선, 중력, 충돌 등 게임 물리의 기본 요소를 직접 체험할 수 있습니다.

- **장르:** 2D 점프 플랫포머
- **목표:** 장애물과 몬스터, 추락을 피해 목표 발판에 도착

---

## 2. 설치 및 실행

### 2-1. 빌드된 실행 파일로 실행 (일반 플레이어)

1. 배포 페이지(예: GitHub Releases) URL_ 에서 최신 빌드를 내려받습니다.
2. 압축을 해제합니다.
3. openSourceTeamProject.exe를 더블 클릭하여 실행합니다.


## 3. 조작법

| 동작 | 키 |
|------|-----|
| 왼쪽 이동 | ← (또는 A)_ |
| 오른쪽 이동 | → (또는 D)_ |
| 달리기 | leftShift |
| 점프 | Space (또는 ↑)_ |

- 점프는 **지면에 있을 때만** 가능하며, 공중에서는 추가 점프가 되지 않습니다. 점프 중에 스페이스바를 떼면 바로 떨어집니다. 

---

## 4. 게임 플레이

### 4-1. 기본 흐름

1. 메인 메뉴에서 게임을 시작하면 레벨을 선택할 수 있습니다.
2. 레벨 선택하면 게임이 시작합니다.
3. 방향키로 이동하고, 점프로 발판을 건너거나 장애물을 넘습니다.
4. 점프하면 자연스러운 **포물선 궤도**를 그리며 떨어집니다.
5. 모든 구간을 통과해 **목표 피자(Goal)** 에 도달하면 클리어됩니다.

### 4-2. 게임 오버

다음 경우 게임 오버가 됩니다.

- 장애물(Obstacle)에 충돌해서 Hp가 0이 될때
- 발판 아래로 떨어져서 Hp가 0이 될때 

게임 오버 시 화면 안내에 따라 **빠르게 재시작**하여 다시 도전할 수 있습니다.

### 4-3. 게임 클리어

목표 발판에 도달하면 클리어 연출이 출력되고, 플레이 데이터가 저장됩니다. 이후 다음 스테이지로 진행 / 그만두기 로 이어집니다.

---

## 5. 메뉴 사용법

메인 메뉴는 다음 기능을 제공합니다.

| 메뉴 | 설명 |
|------|------|
| **게임 시작** | 게임을 시작하고 레벨 선택에 진입합니다. |
| **끝내기** | 게임을 종료할 수 있습니다.. |
